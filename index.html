馬克的潤滑油世界



潤滑學 (Tribology) 是一門綜合性學科，涵蓋了多個領域，包括工程靜力學、動力學、流體力學、熱力學、材料學、燃燒學和應用化學等。儘管大多數學校並未開設專門的潤滑學課程，但實際應用中的經驗對於學習這門學科至關重要。特別是在一些國際知名的潤滑油公司，這些公司不僅擁有研發部門，還有在銷售和服務一線工作的工程師，並與設備製造商長期合作，從而在理論基礎和實踐經驗方面相互印證和傳承。全球性的石油公司，例如 ExxonMobil、Shell、TotalEnergies、BP-Castrol 等，銷售的潤滑油（包括潤滑油脂）涵蓋了工業、海運和航空等多個領域。根據 API 的規範，潤滑油可以根據含硫量、飽和度和黏度指數（VI 指數）的不同分為五大類。基礎油又可以分為礦物油和合成油。在黏度的定義上，我們可以將潤滑油分為引擎油(SAE黏度等級)和非引擎油(ISO黏度等級)。此外，潤滑劑的使用也可以分為液態潤滑油和固態潤滑油。眾多的議題日後會逐一闡述。

因為日益嚴重的溫室效應以及日趨嚴苛的碳排放要求，幾乎所有國際石油公司都開始企業轉型，以永續能源為發展方向，如太陽能、風能、氫能、再生能、等等，石化產業與產品將不再是國際石油公司的主力業務。船舶產業也因國際排放法規要求改用低硫燃油或更乾淨的LNG燃料，更甚者，未來燃料的選項會往乙醇、氨氣或氫氣發展，以2050淨零碳排為最終目標。作為一位多年從事海運潤滑油相關工作的從業人員，對二衝程主機汽缸油等特定議題更感興趣與經驗，這方面的議題討論會多所著墨。如果日後有足夠的時間和資料，我也期待能分享一些基於大數據分析的專業見解，以探討更深入的議題。
